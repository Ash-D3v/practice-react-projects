## ğŸš€ Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Ash-D3v/practice-react-projects.git
   cd practice-react-projects
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```
